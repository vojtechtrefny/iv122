B) L-systémy

Různé obrazce vygenerované pomocí L-systémů. U obarvených variant se barva postupně mění během vykreslování a je tedy možné hezky vidět postup vykreslování.


Úhel: 90°, Základ: "L", Pravidla: "L" → "+RF-LFL-FR+", "R" → -"LF+RFR+FL-"


Úhel: 85°, Základ: "L", Pravidla: "L" → "+RF-LFL-FR+", "R" → "-LF+RFR+FL-"


Úhel: 60°, Základ: "FXF++FF++FF", Pravidla: "F" → "FF", "X" → "++FXF--FXF--FXF++"


Úhel: 60°, Základ: "F", Pravidla: "F" → "+G-F-G+", "G" → "-F+G+F-"


Úhel: 90°, Základ: "F+F+F+F", Pravidla: "F" → "F+f-FF+F+FF+Ff+FF-f+FF-F-FF-Ff-FFF", "f" → "ffffff"


Úhel: 36°, Základ: "F++F++F++F++F", Pravidla: "F" → "F++F++F|F-F++F


Úhel: 25°, Základ: "X", Pravidla: "X" → "F-[[X]+X]+F[+FX]-X", "F" → "FF"
